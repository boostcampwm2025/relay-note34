# 🗓️ Week1 - 우리는 왜 AI를 쓰는가?

## 🎯 토론 주제: "AI를 왜 써야 할까?"  

이번 주 우리는 단순한 도구가 아닌, **AI를 우리가 어떻게 활용하고 왜 써야 하는지** 에 대해 진지하게 토론했습니다.

### 💬 논의된 이유들

- 생산성을 높이기 위해
- 특정 작업의 시작을 빠르게 구상할 수 있음  
- 사람이 처리하기 어려운 **방대한 데이터를 빠르고 정확하게 분석 가능**  
- 개인 맞춤형 분석 및 추천에 강함  
- 반복적이고 대량의 작업을 빠르게 처리 가능  
- 정형화된 문서, 구조, 양식을 빠르게 생성 가능  
- 빠른 의사결정을 도와줄 수 있는 도우미 역할 수행  

---

## 🧭 우리는 어떤 분야에 AI를 쓰기로 했나?

| 분야 | 활용 방식 |
| --- | --- |
| 문서 정리 | 회의록, 요약, 정리 자동화 |
| 아이디어 창출 | 브레인스토밍 파트너로 활용 |
| 아이디어 시제품 | 빠르게 결과물을 만들어보고 피드백 받기 |
| 개인 맞춤 알고리즘 | 나에게 맞는 콘텐츠/정보 제공 |
| 실시간 번역/요약 | 빠르게 정보를 소비하거나 전달할 때 |
| 데이터 분석 | 방대한 양의 자료를 분석하고 핵심만 추출할 때 |
| 즉석 정보 탐색 | "지금 필요한 정보"를 빠르게 받아보기 |

---

## 🎯 이번 주 핵심 질문

> 우리는 AI를 왜, 어디에, 어떻게 써야 유의미하게 나를 도와주는가?  

이에 대한 논의와 결과를 바탕으로 우리는 앞으로 **실험적 퀘스트** 들을 진행하게 됩니다.  
그 첫 시작이 바로 **AI와 함께하는 예술형 퀘스트 (시, 소설, 음악, 미술)** 입니다.
